# Documentation Index

This folder contains detailed documentation for the Riftbound Deckbuilder project, organized by function/area.

## Structure

- **[setup.md](./setup.md)** - Setup and installation instructions
- **[api/](./api/)** - API endpoint documentation
  - [authentication.md](./api/authentication.md) - Authentication endpoints
  - [users.md](./api/users.md) - User management endpoints
  - [decks.md](./api/decks.md) - Deck management endpoints
  - [preferences.md](./api/preferences.md) - User preferences endpoints

## Documentation Guidelines

- Keep documentation separated by function/area
- Each API area should have its own file
- Include request/response examples
- Document validation rules and error codes
- Update documentation when adding new endpoints or features
